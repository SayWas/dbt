# DWH / dbt проект

Проект реализует модельный слой:

- `stg` — очистка и стандартизация данных, пришедших из EL.
- `ods` — инкрементальные снимки цен.
- `dwh` — агрегаты и поведенческие фичи динамики цен.
- `dm` — финальная витрина рекомендаций.

## Команды

```bash
dbt deps --project-dir dwh --profiles-dir dwh/profiles
dbt build --project-dir dwh --profiles-dir dwh/profiles
```

## Важные особенности

- Используются `CTE` и оконные функции (`lag`, `avg over`).
- Настроены `dbt-core` и `elementary` тесты.
- Применены теги для разделения моделей по слоям.
